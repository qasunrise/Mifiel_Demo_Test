# demotests
API, smoke, E2E

## Setup

* Update Java http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
* Clone repo
* Run `npm install`
* Run `node_modules/.bin/selenium-standalone install`

## Run tests
```sh
> selenium-standalone start
```

and in other terminal
```sh
> npm test
```

## Generate docs
```sh
> npm run doc
```
